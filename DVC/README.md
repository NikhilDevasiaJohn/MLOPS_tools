## üì¶ Installation

```bash
# Basic DVC
pip install dvc

# With remote support
pip install dvc[all]        # All remotes
pip install dvc[s3]         # AWS S3
pip install dvc[gdrive]     # Google Drive
pip install dvc[azure]      # Azure Blob

# Verify installation
dvc version
```
